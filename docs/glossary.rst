Glossaire
~~~~~~~~~

.. glossary::

   CORS
      Acronyme de `Cross Origin Resource Sharing
      <https://fetch.spec.whatwg.org/#http-cors-protocol>`_. C'est un protocole
      de couche supérieure au HTTP qui permet au navigateur de vérifier si une
      réponse est partageable avec une `origine
      <https://html.spec.whatwg.org/multipage/origin.html#concept-origin>`_.
      Une mauvaise compréhension du CORS mènera a des problèmes incompris lors
      de l'intégration de l'iframe.
